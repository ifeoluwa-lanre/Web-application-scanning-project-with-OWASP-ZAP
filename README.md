# Web-application-scanning-project-with-OWASP-ZAP
This project demonstrates a comprehensive (DAST) workflow using OWASP ZAP to identify and validate web-based vulnerabilities. The assessment integrates automated scanning with manual security testing to simulate a real-world Application Security audit, ensuring the integrity of web applications within a secure Software Development Life Cycle.


For this project technology used:
- OWASP ZAP; Scanner
- Acunetix; Vulnerable Site
